# SITRAN App

## Variável de ambiente
Crie um arquivo `.env` com:

```env
EXPO_PUBLIC_API_BASE_URL="https://SEU-BACKEND.up.railway.app/api"
```

## Rodar localmente
```bash
npm install
npm start
```

## Observação
Depois que o backend estiver online, o app poderá funcionar por dados móveis ou qualquer Wi‑Fi, sem precisar ficar na mesma rede local do computador.
